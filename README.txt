SAVE THE DAY — V1 PROTOTYPE

How to run:
1. Unzip the folder.
2. Open index.html in a browser.

Included features:
- 80s action title screen
- How to play screen
- 5 timed story rounds
- 12 items per round
- Tap-to-inspect items
- Tap or drag items into the build box
- Assemble tool animation/reveal
- Option A / Option B tool choice
- 1 puzzle point per correct round
- Final 5-question case quiz
- 10-point total score
- Ending rank system

Prototype notes:
- Mobile-friendly tap controls are included because drag-and-drop can be inconsistent on phones.
- No backend required. This is a static HTML/CSS/JS prototype.

V2 updates:
- Added visible item-count requirements for each possible built tool.
- Build Box now shows selected-item progress against the round recipe sizes.
- All 5 story rounds now use 3-minute timers.
